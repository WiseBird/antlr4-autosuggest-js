Getting the antlr-autosuggest JavaScript Module
-----------------------------------------------
```bash
npm install --save antlr4-autosuggest
```

Example
-------

```javascript
// Load the lexer and parser modules generated by antlr4
const myGrammarLexer = require('./myGrammarLexer.js');
const myGrammarParser = require('./myGrammarParser.js');

// Create a grammar factory with the lexer and parser constructors
const autosuggest = require('autosuggest');
const factory = new autosuggest.GrammarFactory(
        myGrammarLexer.myGrammarLexer, myGrammarParser.myGrammarParser);

// Suggest completions for the string "ABC"
var suggestions = new autosuggest.suggest(factory, "ABC");
```

How It Works
------------
See explanation in the [Java project](https://github.com/oranoran/antlr4-autosuggest/).

Building
--------
* Clone the antlr-autosuggest-js repository.
* Install Node and NPM (tested with 8)
* Run the command `npm test`.
